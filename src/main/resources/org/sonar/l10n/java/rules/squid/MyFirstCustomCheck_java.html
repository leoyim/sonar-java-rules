对于具有单个参数的方法，其返回值和其参数的类型永远不能相同

反例

    int     foo3(int value) { return 0; }
    MyClass foo5(MyClass value) {return null; }

正例

    MyClass(MyClass mc) { }
    int     foo1() { return 0; }
    void    foo2(int value) { }
    Object  foo4(int value) { return null; }
    int     foo6(int value, String name) { return 0; }
    int     foo7(int ... values) { return 0;}
